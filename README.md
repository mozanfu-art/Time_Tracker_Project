## Installation & Setup

### Source Code (Developers)
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/time_tracker.git
   cd time_tracker
   ```
2. Ensure Flutter SDK is installed and configured:
   ```bash
   flutter doctor
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```

---

### Android Users
- Download the APK installation file from the attached **Google Drive link**.  
- ‚ö†Ô∏è **Note:** The APK file is only for **Android devices**.

  1. Open **Settings ‚Üí Security**.
  2. Enable **Install from unknown sources** (if prompted).
  3. Tap the APK file to install.  
- Launch the app and start tracking your time.
